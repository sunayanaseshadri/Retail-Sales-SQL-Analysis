# Retail Sales Performance Analysis Using SQL

## Project Overview
This project analyzes retail sales data to uncover revenue trends, customer behavior, product performance, and profitability insights. The objective is to generate data-driven recommendations to support pricing, marketing, inventory, and customer retention strategies.

## Tools Used
- SQL (MySQL / PostgreSQL / SQLite)
- GitHub
- Optional: Excel / Power BI / Tableau for visualization

## Business Questions Answered
- What are the overall revenue and profit trends?
- Which products and categories generate the most revenue?
- Who are the most valuable customers?
- How does performance vary by region and shipping method?
- Which products have low profitability and need optimization?

## Key Insights
- Identified top 10 revenue-driving products and categories.
- Discovered high-value customer segments with repeat purchase behavior.
- Flagged low-margin products impacting overall profitability.
- Analyzed shipping methods affecting delivery time and revenue.
- Highlighted regional performance differences.

## Business Recommendations
- Prioritize promotions for high-margin and high-demand products.
- Optimize pricing or sourcing strategies for low-profit items.
- Improve logistics performance for slower shipping modes.
- Implement loyalty programs for top customer segments.
